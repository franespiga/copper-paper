\section{Integration limitations log}
\label{app:limitations-log}

The living markdown file \texttt{MANUSCRIPT\_LIMITATIONS.md} records missing outputs,
template gaps, Overleaf workflow constraints, and blocking issues for the manuscript layer.
Scientific limitations are summarized in Section~\ref{sec:limitations}; this appendix points
reviewers and collaborators to the full integration log for build and submission readiness.

The manuscript is labeled a \textbf{draft automation scaffold} until:
\begin{itemize}
  \item all required results exist and are synced;
  \item \texttt{make paper-pdf} compiles locally;
  \item the target journal is frozen; and
  \item every figure and table is traceable to code-generated outputs.
\end{itemize}
